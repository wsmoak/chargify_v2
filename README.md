ChargifyV2
==========

** Chargify API V2 (Chargify Direct) in Elixir **

## Usage

set environment variables for
`CHARGIFY_DIRECT_API_KEY` and `CHARGIFY_DIRECT_PASSWORD`

$ iex -S mix
> ChargifyV2.get!("/calls/<CALL_ID>")
